## 개요
리그오브레전드의 `학교별 랭킹!` 을 보여주는 서비스 입니다.

### FrontEnd
* React.js/Typescript
* Redux/ Redux-toolkit

### BackEnd
* Express.js
* mariaDB
* Riot API
* TypeORM

## 필요 목록
개발 하면서 필요한 기능에 대한 정리 부분입니다.
### FrontEnd Pages
* LoginModal: 로그인 버튼 클릭하면 등장하는 Modal 컴포넌트 
  - Modal 이벤트
  - 모달 로딩시 Screen Mask 적용, 따로 컴포넌트를 만들어서 사용해야할듯(자체 컴포넌트 css로 하면 부자연스러움)
    
* Home: 홈페이지에서 랭킹 정보를 출력할 예정 
  - 표 형식
  - 한 페이지에 1 ~ 20명 출력 넘어가면 pagination
  - 검색으로 몇 위에 있는지 검색 하도록 구현
* 회원가입 페이지
* 소셜 회원가입

### BackEnd
* JWT && PassPort를 이용하여 로그인 
* nodemailer 를 이용하여 이메일인증, 회원가입 구현예정
* 학교 정보가져오기(검색)
