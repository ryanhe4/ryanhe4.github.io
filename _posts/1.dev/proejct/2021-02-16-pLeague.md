## 개요
리그오브레전드의 `학교별 랭킹!` 을 보여주는 서비스 입니다.

### FrontEnd
* React.js/Typescript
* Redux/ Redux-toolkit
* react-query

### BackEnd
* Express.js
* mariaDB
* Riot API
* career open API  
* TypeORM
* nodemailer
